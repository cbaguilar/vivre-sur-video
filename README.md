# Arduino Music Project

Current specs: Arduino Nano IoT. One of the many microcontrollers that I have acquired for free over the years. People are so kind.



# Goals:

- Make a chiptune synthesizer inspired by the Atari POKEY: https://en.wikipedia.org/wiki/POKEY
- Wire up an amplification circuit
- Set up a non-painful track/sequencing mechanism
- Add pretty lights
